Common CSS Properties Cheatsheet

Colors in CSS

CSS Declaration What it Does
Color Sets Foreground Color
Background-color Sets Color of Element's Container
RGBA (red #, green #, blue #, transparency%)
#RRGGBB 00 (lowest value) - FF (highest value)
Opacity Values: 0 (transparent) - 1 (opaque)

Fonts in CSS

CSS Declaration What it Does
Font-family Contains many fonts, use as fallback
Font-style Mainly to italicize text
Font-weight Specifies weight of a font
Font-size Sets font size, can be absolute or relative
Google Fonts Wide array of fonts; requires <link> in HTML's <head>

Borders in CSS

CSS Declaration What it Does
Border-width Specifies width of the four borders
Border-style Specifies what kind of border to display
Border-color Sets color of all four borders
Border-top-style Sets style for top border (can be used on bottom, left, right)
Border-radius Used to add rounded borders to an element

Selectors in CSS

CSS Declaration What it Does
body Styles all <body> elements
.important Styles all elements with class="important"
#crucial Styles all elements with id="crucial"
p .important Styles all <p> elements with class="important"
li, p Styles all <li> and <p> elements
